¿Cómo hacer una cortina hawaiana?

Mide el marco de puerta o entrada donde deseas colocar la cortina hawaiana

Las cortinas hawaianas se fabrican midiendo la zona donde se van a colocar, y asegurándose de que tanto el marco como el material alrededor son lo suficientemente fuertes como para soportar su peso. Esto suele significar colocarlas en marcos de hormigón o concreto. En caso de ser tabla roca, recomendamos la instalación adicional de un marco de metal para distribuir el peso y soportar el movimiento prolongado de las hawaianas.

Elige bien el PVC para tu cortina hawaiana armada

Si necesitas que el PVC para cortina hawaiana soporte el paso de montacargas, recomendamos utilizar siempre nuestros modelos con refuerzo de costilla o hilo en nylon. Si necesitas que soporte temperaturas muy bajas, también podemos ayudarte con un PVC para cuartos fríos o cámaras de refrigeración. También tenemos anti insectos, anti estática, para áreas de soldadura entre muchas otras aplicaciones.

Para comenzar a armar tu cortina hawaiana, deberás considerar:

  1. Cada tira de PVC deberá tener lainas distribuidas uniformemente para que no se maltraten al colgarlas. Las tiras deberán ser cortadas uniformemente para mantener el peso distribuido uniformemente.
  2. El perfil que va atornillado al marco de tu entrada, deberá ser compatible con las lainas para tener movilidad completa con el mínimo desgaste
  3. El perfil deberá ser de un material que soporte el peso y movimiento constante de las hawaianas, además de ir muy bien atornillado para evitar que se pandee o se doble con el uso
